Home Terminal Commands Code snippets
Terminal
'cd' - change directory to enter the folder you need to be in
'ls' - short for 'list' and lists items in currenct directory
'mkdir' - short for make directory and creates a new folder
'code .' - opens the currenct directory in VScode
Node
'node' - starts the nodejs interpreter
'node filename.js' - executes a js file
'npm' - package management, e.g. installing nodemon: npm install nodemon
'nodemon' - used to restart a node server automatically on changes
Git
'git clone url' - downloads a repository to current directory
'git add file(s)' - moves local changes to staging area
'git commit -m' - takes the staged changes and commits it to the project history
'git push' - pushes the committed changes to online repository
'git pull' - downloads last updated version of a repository
